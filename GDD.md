## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| 제목 | 던전왕이 될 거야 |
| 장르 | 캐주얼 로그라이크 RPG |
| 플랫폼 | iOS / Android |
| 엔진 | Godot 4.x |
| 조작 | 터치 기반 |
| 목표 | 30층 던전 최하층 생존 클리어 |
| 핵심 재미 | 매 런 다른 경험 + 죽어도 성장이 남는 쾌감 |
| 코어 컨셉 | 던전왕을 꿈꾸는 겁쟁이의 죽기 싫은 생존기 |
| 아트 스타일 | 손그림 두들 (밝고 귀여운 카툰) |

## 2. 핵심 게임 루프

```
[시작] → 맵 탐험 → 전투 → 아이템 수집 → 자원 관리
   ↓                                              ↓
[클리어] ←── 보스 처치 ←── 층 하강 ←── 레벨업/강화
   ↓
[사망 시] → 경험치 획득 → 룬/제작법 해금 → 다음 런 강해짐
```

## 3. 맵 시스템

### 층 구조

- **총 30층**, 매 5층마다 보스
- 보스 처치 후 다음 구간 진입

| 구간 | 층 | 보스 |
|------|----|------|
| 1구간 | 1~5층 | 보스 A |
| 2구간 | 6~10층 | 보스 B |
| 3구간 | 11~15층 | 보스 C |
| 4구간 | 16~20층 | 보스 D |
| 5구간 | 21~25층 | 보스 E |
| 6구간 | 26~30층 | 최종 보스 |

### 맵 생성 규칙

- **절차적 생성** — 매 런마다 완전히 다른 맵
- 안개 효과 — 탐험한 구역만 시야 확보
- 방, 복도, 특수룸(상점, 캠프파이어, 함정 등) 배치

### 특수 오브젝트

| 오브젝트 | 기능 |
|----------|------|
| 캠프파이어 | HP 회복 |
| 상점 | 골드로 아이템 구매 |
| 함정 | 밟으면 피해 |
| 보물상자 | 랜덤 아이템 |
| 계단 | 다음 층 이동 |

## 4. 직업 시스템

### 해금 조건

| 직업 | 해금 조건 | 특성 |
|------|-----------|------|
| **전사** | 기본 해금 | 높은 체력, 근접 물리 특화 |
| **마법사** | 6층 도달 | 정신력 기반, 광역 마법 |
| **도적** | 11층 도달 | 은신, 치명타, 민첩 특화 |
| **사냥꾼** | 16층 도달 | 원거리, 투척 특화 |

### 직업별 스킬 구조

- **무기 스킬**: 쿨타임 3턴, 장착 무기 연동
- **직업 고유 스킬**: 쿨타임 5턴

| 직업 | 고유 스킬 |
|------|-----------|
| 전사 | 방어 태세 / 강타 |
| 마법사 | 광역 번개 / 마법 방벽 |
| 도적 | 은신 + 주변 광역 데미지 + 실명 (20AP 동안 은신) |
| 사냥꾼 | 전방위 6발 발사 (적 수 무관, 1발씩 / 잔탄 1개 소모) |

## 5. 스탯 시스템

| 스탯 | 영향 |
|------|------|
| **힘 (STR)** | 물리 공격력, 장비 착용 조건 |
| **체력 (CON)** | 최대 HP |
| **민첩 (DEX)** | 명중률, 회피율, 행동 속도 |
| **지능 (INT)** | 마법 공격력, 최대 정신력 |
| **운 (LUK)** | 크리티컬, 아이템 드랍률 |

> **힘 부족 페널티**: 요구 힘 초과 장비 착용 → 명중 / 회피 / 행동속도 감소

## 6. 전투 시스템

### 턴 구조

- 1턴 = **20 AP**
- 대기 버튼 1회 = 10 AP 소모
- 이동·공격 모두 AP 소모

### 몬스터 행동 속도

| 속도 | AP |
|------|----|
| 빠름 | 10 AP 미만 |
| 보통 | 10 AP |
| 느림 | 20 AP |

### 전투 흐름

1. 터치로 이동
2. 몬스터 감지 시 캐릭터 위 `!` 표시
3. 몬스터 터치 → 공격
4. 보스 스킬 사전 경고 → 붉은 점멸 1턴 → 스킬 발동
5. 보스 스킬은 1칸 후퇴로 회피 가능

### 공격 타입

- 근거리 / 원거리
- 물리 / 마법(번개)

## 7. 생존 자원 관리

| 자원 | 고갈 효과 | 회복 방법 |
|------|-----------|-----------|
| **HP** | 0 → 사망 (퍼마데스) | 물약, 캠프파이어, 음식 |
| **배고픔** | 일정 수치 초과 → HP 틱 감소 | 음식 섭취 |
| **피로도** | 일정 수치 초과 → 정신력 감소 | 휴식·수면 |
| **정신력** | 마법 사용 불가 | 정신력 물약, 휴식 |

## 8. 아이템 시스템

### 아이템 카테고리

| 카테고리 | 특징 |
|----------|------|
| **장비** (무기/방어구) | 힘 요구치, 강화 가능, 저주/축복 상태 |
| **물약** | 정체불명 시스템 (런마다 색/효과 랜덤) |
| **주문서** | 1회용, 고대 주문서는 배우면 영구 사용 |
| **음식** | 배고픔 회복 |
| **재료** | 제작에 사용 |
| **씨앗** | 특수 효과 |
| **원거리 무기** | 별도 슬롯 운용 |

### 정체불명 시스템 (물약)

- 매 런마다 색깔-효과 랜덤 매핑
- 마셔보거나 감정해야만 효과 확인 가능
- 런 내에서 한 번 확인하면 이후 동일 색 = 동일 효과

### 장비 상태

- **저주 장비**: 해제 전까지 탈착 불가, 능력치 하향
- **축복 장비**: 능력치 상향
- **강화**: 주문서로 강화 → 성공 시 내구도 풀 회복

### 인벤토리 / 가방

| 가방 종류 | 용도 |
|-----------|------|
| 재료 가방 | 제작 재료 수납 |
| 주문서 가방 | 주문서 수납 |
| 연금술 가방 | 물약 재료 수납 |
| 보조 가방 | 범용 (효율 최고) |

## 9. 제작 시스템

| 제작 종류 | 재료 | 결과물 |
|-----------|------|--------|
| 연금술 | 빈 병 + 시약 재료 2개 | 물약 |
| 아티팩트 제작 | 제작법 + 재료 수집 | 강력한 장비 |
| 분해 | 장비 | 제작 재료 |

## 10. 메타 진행 시스템 (런 간 지속)

### 경험치 (원작명: 기억의 티끌)

- 보스 처치 / 사망 시 획득
- 다음 런에 이월되어 룬·제작법 해금에 사용

### 룬 시스템

| 항목 | 내용 |
|------|------|
| 종류 | 직업별 룬 + 모험가 룬 (총 5종류) |
| 등급 | 기본 → 일반 → 희귀 → 에픽 → 레전드 |
| 장착 슬롯 | 최대 5개 (층 진행으로 순차 해금) |
| 레벨업 | 동일 룬 중복 수집 + 고대 주화 소모 |
| 최대 레벨 | 일반 60 / 희귀 35 / 에픽 12 / 레전드 5 |

### 제작법

- 한 번 해금 → 이후 런에 영구 보유
- 게임 중 재료 수집 → 아티팩트 제작 가능
