# 던전왕이 될 거야 — 로드맵

> 벤치마킹: [던전에서 죽기싫어](benchmarking/던전에서-죽기싫어.md)

---

## 게임 개요

모바일 턴제 로그라이크. 854×480 landscape.  
절차적 던전 탐험 → 재료 수집 → 제작 → 더 깊은 층으로.  
총 25층. 계단은 위로만 (돌아올 수 없음).

## 층 구조

| 층 | 타입 |
|----|------|
| 3, 8, 13, 18, 23층 | 🛒 상인 |
| 5층 | 👑 슬라임 킹 |
| 10층 | 👑 고블린 족장 |
| 15층 | 👑 스켈레톤 마법사 |
| 20층 | 👑 뱀파이어 군주 |
| 25층 | 👑 어둠의 드래곤 (최종) |
| 나머지 | 일반 |

보스 처치 → 열쇠 드롭 → 열쇠 있어야 계단 이동 가능

## 구현 단계

### Phase 1 — 현재 구현됨 ✅
- 절차적 맵 생성
- 플레이어 이동 + 전투 (공격/피격)
- 몬스터 AI (추적 + 공격)
- 독/화염 상태이상 (플레이어 + 몬스터)
- 층별 몬스터 타입 다양화 (슬라임/고블린/스켈레톤/유령)
- 물약 정체불명 시스템 (4색 4효과)
- 물약 던지기
- 레벨업 시스템
- 층 이동 (페이드 인/아웃)
- HUD (HP바, 인벤 슬롯, 스킬 슬롯, 로그)
