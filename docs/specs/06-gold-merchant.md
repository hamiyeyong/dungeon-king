## 골드 시스템

### 획득
| 출처 | 골드 |
|------|------|
| 몬스터 처치 | 층 × 랜덤(2~5) |
| 보물상자 | 10~30골드 랜덤 |
| 아이템 판매 (상인) | 구매가의 50% |

### 표시
- HUD 좌측 상단 또는 하단에 💰 N골드 표시

---

## 상인층 (3, 8, 13, 18, 23층)

- 상인 NPC가 맵에 등장
- 플레이어가 인접하면 상점 UI 오픈

### 상점 구성
매 상인층마다 랜덤 6~8개 아이템 진열

### 가격표
| 아이템 | 구매가 | 비고 |
|--------|--------|------|
| 미확인 물약 | 30골드 | 뭔지 모르니 싸게 |
| 회복 물약 (식별됨) | 80골드 | |
| 포만 물약 (식별됨) | 60골드 | |
| 독/화염 물약 (식별됨) | 50골드 | |
| 음식 | 40골드 | |
| 주문서 (미확인) | 50골드 | |
| 주문서 (식별됨) | 120골드 | |
| 나무 방패 | 80골드 | |
| 철 방패 | 200골드 | |
| 천 갑옷 | 80골드 | |
| 가죽 갑옷 | 180골드 | |
| 철검 | 250골드 | |
| 재료 (나뭇가지/돌/천) | 15골드 | |
| 열쇠 | 판매 불가 | 버리기도 불가 |

### 판매 규칙
- 인벤 아이템을 상인에게 팔 수 있음 (구매가의 50%)
- 열쇠는 판매/버리기 불가
