## 맵 오브젝트

### 🌿 수풀 (Grass)
```
배치    바닥 타일의 약 2.5% 무작위 배치
상호작용  공격하면 소멸 + 아이템 1개 드롭
드롭 테이블
  나뭇가지  30%
  약초      25%
  돌        20%
  빈병      15%
  물약       7%  (미확인 상태)
  주문서     3%  (미확인 상태)
특이사항  밟아도 아무 일 없음. 공격해야 수집 가능
```

### 🏺 항아리 (Jar)
```
배치    바닥 타일의 약 2% 무작위 배치
상호작용  공격하면 파괴(JAR_OPEN) + 결과 발동
결과
  40%  골드 또는 아이템 드롭 (천/나무/물약 등)
  30%  빈 항아리 (아무것도 없음)
  20%  몬스터 1마리 즉시 소환
  10%  저주 상태이상 부여
```

### 🔥 야영지 (Campfire / Campfire_Out)
```
배치    각 층 룸 중앙에 1개 (CAMPFIRE 상태로 시작)
점화 재료  꺼진 야영지(CAMPFIRE_OUT) → 나뭇가지 ×1 또는 횃불 ×1 소모 → 재점화
기능    [요리] 식량 → 익은 식량, 생고기 → 구운 고기 (여러 번 가능, 불 유지)
        [휴식] 피로도 초기화 + HP +5 (1회만, 이후 CAMPFIRE_OUT)
```

### ⚗️ 연금술 솥 (White/Black Cauldron)  ← 추가 예정
```
배치    각 층 1개, 흰 솥 또는 검은 솥 중 하나 랜덤 배치
상호작용  인접 이동 시도 → 물약 제작 팝업
기능    [흰 솥] 빈병 + 재료 2개 → 물약 (발현도 높은 재료 기준)
        [검은 솥] 빈병 + 재료 2개 → 물약 (발현도 낮은 재료 기준)
특이사항  사라지지 않음 (무한 사용 가능)
```

### 🌀 이상한 우물 (Magic Well)  ← 추가 예정
```
배치    층당 0~1개 (낮은 확률)
상호작용  아이템을 던지면 변환
결과
  50%  같은 종류의 다른 아이템으로 변환
  30%  강화 +1
  15%  축복 부여
   5%  아이템 소멸 또는 저주
```

### 🪤 함정 (Trap)
```
배치    각 층 랜덤 2~4개 (복도에 주로 배치)
상태    미발각 → 발각 → 발동 후 비활성 (단, 스파이크는 계속 발동)
외관    미발각: 바닥처럼 보임 (FOV 내 가까이 오면 아이콘 표시)
        발각됨: 위험 아이콘 표시
상호작용  밟으면 즉시 발동. 1턴 소모
해제     아무 아이템이나 던져서 원격 발동 → 해제 가능
         (독구름/마비가스 함정은 더 멀리서 던져야 안전)
```

**함정 종류 (현재 구현 3종 + 추가 예정 7종)**

| 함정 이름 | 구현 | 효과 |
|---------|------|------|
| 독가시 | ✅ | HP -5 + 독 상태이상 |
| 화염 | ✅ | HP -8 + 화염 상태이상 + 인접 1칸 AOE |
| 텔레포트 | ✅ | 층 내 랜덤 위치로 이동 |
| 스파이크 | ← 추가 예정 | 밟으면 가시 돌출, 데미지. **비활성화 안 됨 (계속 발동)** |
| 부상 | ← 추가 예정 | 부상 상태이상 (행동 속도 저하 + 지속 HP 감소) |
| 독구름 | ← 추가 예정 | 발동 시 가로/세로 2칸, 대각선 1칸 AOE 독가스 살포. 원거리 해제 필수 |
| 마비가스 | ← 추가 예정 | 발동 시 가로/세로 4칸, 대각선 1.5칸 마비가스 살포. 1회 피격 후 면역 |
| 얼음 | ← 추가 예정 | 빙결 상태이상. 날고기를 던져 얼린 고기 제작 가능 |
| 소환 | ← 추가 예정 | 적 1~3마리 즉시 소환 |
| 알람 | ← 추가 예정 | 층 내 인근 적들에게 플레이어 위치 알림 |

발각 조건:
- 인접(거리 1)하면 자동 발각 (발동 안 함, 아이콘만 표시)
- 횃불 보유 시 시야 확장으로 더 일찍 발각 가능
- 식별 주문서 사용 시 층 전체 함정 발각
- 탐색 모드 활성화 시 인접 타일 숨겨진 요소 확정 발각 (AP 추가 소모)

### 🌊 오염된 샘 (Tainted Spring)  ← 추가 예정
```
배치    층당 0~1개
상호작용  마시면 체력 회복 or 독 상태이상 (50/50)
          정화 물약을 던지면 → 맑은 샘으로 변환
```

### 💧 맑은 샘 (Clear Spring)  ← 추가 예정
```
배치    오염된 샘에서 변환 또는 드물게 자연 배치
상호작용  마시면 HP·MP 회복 + 배고픔·피로도 -250 + 상태이상 해제
          저주받은 아이템을 던지면 저주 해제
특이사항  포션을 물에 던지면 효과 확인 가능 (단, 한 샘에 같은 물약 여러 번 던져도 1번만)
```

### 💀 해골더미 (Skull Pile)  ← 추가 예정

```
배치    층당 0~2개
상호작용  랜덤 아이템 확정 드롭 (장비/재료/포션 중 1개)
특이사항  통과 불가 (돌아가야 함)
```

```
큰 제단   더 좋은 아이템 배치
```

### 📜 지식의 석판 (Knowledge Tablet)  ← 추가 예정

### 👼 천사 석상 (Angel Statue)  ← 추가 예정

### 🌿 약초밭 (Herb Patches)  ← 추가 예정

약초밭은 밟으면 효과 발동 + 일정 확률로 재료 아이템 드롭.
씨앗으로 해당 밭을 인위적으로 생성 가능.

| 밭 종류 | 밟을 때 효과 | 드롭 아이템 |
|---------|------------|-----------|
| 얼음송이 | 데미지 없는 빙결 | 식은 얼음송이 |
| 피이끼 | 데미지 + 마비 | 말린 피이끼 |
| 산삼 | HP 회복 | 산삼 뿌리 |
| 나이트쉐이드 | 데미지 + 독가스 살포 (범위, 지속) | 나이트쉐이드 잎 |
| 암브로시아 | 40AP간 무적 필드 전개 (넉백 공격에 취약) | 암브로시아 꽃 |
| 영지버섯 | HP 재생 | 말린 영지버섯 |
| 만드라고라 | MP 회복 (드물게 데미지) | 만드라고라 뿌리 |
| 화염초 | 불붙음 상태 | 화염초 꽃잎 (야영지 점화 재료) |
| 꿈결초 | 수면 상태 + 소량 데미지 → 피로도 감소 | 꿈결초 꽃잎 |

> 공중 판정 적(박쥐류)은 함정은 무시하지만 약초밭은 밟는다.

### 📦 보물상자 (Chest)

```
내용물
  슬롯 A (필수)  포션 or 음식
  슬롯 B (필수)  장비 or 재료
```

## 문 시스템 (Door)

```
방 입구마다 문 배치 (선택적)
상태    닫힘 → 열림 (한번 열면 유지)
상호작용  문에 인접한 상태에서 이동키 → 1턴 소모하며 개방
시야    닫힌 문 → 너머 시야 차단
        열린 문 → 시야 통과 가능
몬스터  인간형만 문 열 수 있음 (추후)
```

## 시야 시스템 (Fog of War)

```
전체 맵    기본적으로 어두움 (검은 오버레이)
시야 반경  플레이어 중심 N칸만 밝혀짐

기본 시야   반경 4칸
횃불 보유   반경 7칸 (인벤에 횃불 있을 때)

닫힌 문     시야 차단 (문 너머 안 보임)
열린 문     시야 통과
```

### 탐색된 영역
- 한 번 밝혀진 칸은 어둡게 표시 (완전 검정 아님, 흐릿하게)
- 현재 시야 내: 완전히 밝음
- 과거 탐색 영역: 흐릿하게 (몬스터 위치 미표시)
- 미탐색: 완전히 어둠

## 층 이동

```
계단 (STAIRS) — 마지막 방 중앙
  밟으면 팝업 표시:
    "다음 층으로 올라갈까요?\n(다시 돌아올 수 없습니다)"
  확인 → 페이드 아웃 → 다음 층 생성 → 페이드 인
  취소 → 팝업 닫힘, 몬스터 턴 진행
```

한 방향(위)으로만. 한 번 올라가면 이전 층으로 되돌아갈 수 없음.

### 층 이동 시 유지
- 플레이어 HP, MP
- 배고픔, 피로도
- 인벤토리 아이템
- 장착 장비

### 층 이동 시 리셋
- 맵 (새로 절차적 생성)
- 몬스터 배치
