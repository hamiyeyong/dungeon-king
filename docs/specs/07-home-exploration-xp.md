# Spec: 홈 화면 & 탐험경험치 시스템

> Status: 구현 중
> Priority: Phase 2

## 개요

게임 시작 전 홈 화면을 추가하고, 런을 거듭할수록 탐험경험치(탐험 XP)가 누적되어
일정량 이상 쌓이면 영구 보너스 보상이 해금되는 메타 진행 시스템.

## 관련 씬/노드

`Home` · `SaveData` · `Main` · `HUD`

- [[02-monster-combat]] — 층 이동/보스 클리어 XP 트리거

## 씬 흐름

```
홈 화면 (home/home.tscn)
  └─ "게임 시작" 클릭 → main/main.tscn (로드)
       └─ 게임 오버 / 게임 클리어 → 홈 화면으로 복귀
```

- 앱 최초 실행 → 홈 화면 진입
- 게임 오버 / 클리어 화면의 버튼 라벨: **"홈으로"** (기존 "다시 시작" 제거)
- 홈으로 돌아올 때 탐험 XP를 자동 저장 (user://save_data.cfg)

---

## 탐험경험치 (탐험 XP)

런 종료(사망·클리어) 시 해당 런에서 획득한 XP가 누적 저장된다.

### 획득 방법

| 행동 | XP |
|------|-----|
| 계단으로 다음 층 이동 | 현재 층 × 10 XP |
| 물약 정체 최초 확인 (사용/감정) | 50 XP |
| 아이템 제작 성공 (미구현 — 추후 연동) | 40 XP |

예시 런:
- 3층까지 생존 후 사망 → (10+20) = 30 XP
- 25층 풀 클리어 → 10×(1+2+…+25) = 3,250 XP (+ 물약 최대 300 XP)

### 저장소

| 항목 | 내용 |
|------|------|
| 저장 파일 | `user://save_data.cfg` |
| Autoload 이름 | `SaveData` |
| 저장 시점 | 게임 오버 / 클리어 직후 |

## 마일스톤 보상 (영구 누적)

| 인덱스 | 필요 XP | 보상 | 타입 |
|--------|---------|------|------|
| 0 | 200 | 최대 HP +2 | max_hp |
| 1 | 500 | 최대 MP +5 | max_mp |
| 2 | 1,000 | 물약 1개 감정된 상태로 시작 | identify_potion |
| 3 | 2,000 | 최대 HP +3 추가 (누적 +5) | max_hp |
| 4 | 3,500 | 치유 물약 1개 지참 시작 | start_potion_heal |
| 5 | 5,000 | 공격력 +1 | atk |

- 모든 보상은 **영구 적용** — 한 번 해금되면 그 이후 모든 런에 자동 적용
- 홈 화면에서 잠금/해금 상태 및 진행도 바 표시
