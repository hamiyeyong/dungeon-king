# Spec: 아이템 시스템 (Item System)

> Status: 작성 중
> Priority: Phase 1 MVP

---

## 개요

인벤토리 10칸 단일 관리. 소모품/장비/주문서/재료를 모두 여기서 관리.
물약은 정체불명 시스템으로 런마다 색-효과가 랜덤 매핑된다.

## 관련 씬/노드

`Player` · `HUD` · `Map`

## 인벤토리

- **슬롯 수**: 20칸 (소모품 + 장비 미장착분 + 재료 통합)
- 아이템 줍기: 플레이어가 아이템 타일로 이동 시 자동 획득 (꽉 차면 미획득)
- 층 이동 시 유지, 게임 오버 시 초기화

## 장비 슬롯

```
[무기] [방패] [갑옷] [투척]  ← HUD 별도 표시, 인벤 차지 안 함
```

### 투척 슬롯

| 아이템 | 묶음 수 | 데미지 | 사용 방법 |
|--------|---------|--------|----------|
| 다트 | 5발 | 3 | 인벤 "투척 장착" → 투척 슬롯 탭 (활 불필요) |
| 나무 화살 | 10발 | 5 | 인벤 "투척 장착" → 투척 슬롯 탭 (활 필수) |

- 다트: 인벤에서 "투척 장착" → 투척 슬롯에 장착 → 하단바 슬롯 탭으로 타겟팅
- 나무 화살: 투척 슬롯 탭 시 **활** 미장착이면 "활을 장착해야 화살을 쏠 수 있습니다."
- 묶음 소진 시 슬롯 자동 비워짐

직업: **전사 고정** (클래스 선택 없음)

### 장비 내구도

```
무기   몬스터 공격 시 내구도 -1 (수풀/환경 공격은 소모 없음)
방패   피격(막기) 시 내구도 -1
갑옷   피격 시 내구도 -1

내구도 1 이상  → 정상 기능
내구도 0       → ATK/DEF 기여 절반으로 감소 (파괴되지 않음)
표시  "[목검 3/15]" 형태로 인벤/HUD에 표시

수리 방법:
  수리도구 사용    → 해당 장비 내구도 절반 회복 + 최대 내구도 소폭 감소(-1)
                    (여러 번 사용 가능하나 최대 내구도가 점점 줄어듦)
  상인 수리 서비스 → 전체 내구도 완전 회복 + 최대 내구도 유지 (골드 소모)
```

### 강화 두루마리

```
사용 → 장착 무기 강화 +1 + 최대 내구도 완전 회복
최대 +10강. 이미 +10이면 사용 불가
제작: 약초 ×2 + 나뭇가지 ×1
```

## 주문서 (스킬 슬롯)

- HUD 스킬 슬롯 2개에 장착
- 탭 → 발동 → MP 소모 → 소멸 (일회용)
- MP 부족 시 HP로 대신 발동
- 미확인 상태로 드롭 → 사용 시 정체 공개 (물약과 동일)

| 분류 | 주문서 | 효과 |
|------|--------|------|
| 공격 | 강타 주문서 | 다음 공격 3배 데미지 |
| 생존 | 순간이동 주문서 | 층 내 랜덤 위치로 이동 |
| 유틸 | 식별 주문서 | 인벤 미확인 아이템 전부 식별 |

## 소모품

### 포션 (10종)

색상 10종 (빨강/파랑/초록/노랑/보라/흰색/주황/분홍/하늘/갈색) ↔ 포션 10종 랜덤 배정.

| 포션 | 효과 (마실 때) | 효과 (던지기) | 식별 조건 | 주재료 |
|------|--------------|--------------|----------|--------|
| 회복 물약 | HP +max_hp×80% 즉각 | 깨짐 | 마셔야 공개 | 산삼 뿌리 |
| 재생 물약 | 5턴간 HP 재생 (~max_hp×40%) | 깨짐 | 마셔야 공개 | 말린 영지버섯 |
| 정신력 물약 | MP +max_mp×80% 즉각 | 깨짐 | 마셔야 공개 | 만드라고라 뿌리 |
| 독 물약 | HP -5, 독 상태이상 | 적 HP -8 + 독 부여 | 던질 때 공개 | 나이트쉐이드 잎 |
| 화염 물약 | HP -8, 화염 상태이상 | 적 HP -10 + AOE HP -5 | 던질 때 공개 | 화염초 꽃잎 |
| 정화 물약 | 모든 상태이상 & 저주 해제 | 적의 상태이상 제거 | 마셔야 공개 | 생마늘 |
| 수면 물약 | 수면 3턴 + 피로도 -100 | 적 수면 3턴 | 던질 때 공개 | 꿈결초 꽃잎 |
| 마비 물약 | 마비 3턴 | 적 마비 3턴 | 던질 때 공개 | 말린 피이끼 |
| 얼음 물약 | 느려짐 3턴 | 적 느려짐 3턴 | 던질 때 공개 | 식은 얼음송이 |
| 경험치 물약 | 경험치 +150 | 깨짐 | 마셔야 공개 | 암브로시아 꽃 |

### 음식
| 음식 | 배고픔 감소 | 획득 방법 |
|------|------------|----------|
| 식량 | -30 | 상자, 바닥 드롭 |
| 익은 식량 | -60 | 식량을 모닥불에 던지면 생성 |

- 식량/익은 식량은 던져도 깨지지 않고 목표 타일에 떨어짐
- 식량을 모닥불(CAMPFIRE) 타일에 던지면 → 모닥불 위에 익은 식량으로 변환

### 모닥불 상호작용
```
모닥불(CAMPFIRE) 방향으로 이동 시도 → "야영 / 요리 / 취소" 3버튼 팝업

[야영]: 화면 페이드 아웃 → 피로도 0 회복 + HP +5 → 페이드 인 → 모닥불 꺼짐(CAMPFIRE_OUT)
[요리]: 인벤에서 생고기 선택 → 익은 식량으로 변환 (모닥불 유지, 적 턴 진행)
        - 생고기가 없으면 "요리할 생고기가 없습니다." 로그
        - 식량은 요리 불가
[취소]: 적 턴 진행 (이동 없음)

꺼진 모닥불(CAMPFIRE_OUT) 방향으로 이동 시도 → "불을 피울까요?" 팝업
확인: 나뭇가지×1 또는 횃불×1 소모 → CAMPFIRE 복원
      - 재료 없으면 팝업 없이 "재료가 없습니다." 로그
취소: 적 턴 진행
```

## 정체불명 시스템 (물약)

```
런 시작 시: 빨강/파랑/초록/노랑/보라/흰색/주황/분홍/하늘/갈색 → 10종 포션 랜덤 배정

식별 전 표시: "빨간색 물약", "파란색 물약" (xx색 물약 형태)
식별 후 표시: "회복 물약", "독 물약" (효과명)

식별 방법:
  사용/음주                      → 효과 발동과 함께 공개
  던지기 (독/화염/수면/마비/얼음) → 던지는 순간 공개
  식별 주문서                    → 전체 공개
```

수면 상태이상:
```
적용 시 N턴간 행동 불가 (이동/공격 못 함)
공격받으면 즉시 해제
플레이어에게 걸릴 경우 N턴간 입력 무시
```

## 재료

| 재료 | 획득처 | 용도 |
|------|--------|------|
| 나뭇가지 | **수풀** 공격 | 무기/방패 제작, 강화 두루마리 제작 |
| 약초 (미식별) | **수풀** 공격 (확률) | 연금술 솥 물약 재료, 강화 두루마리 제작, **직접 섭취 (랜덤 효과)** |
| 돌 | **수풀** 공격 (확률) | 무기/방어구 제작 |
| 천 | **항아리** 파괴 (확률) | 방어구 제작 |
| 광석 | **장비 분해** | 수리도구 제작 |
| 독초 | 수풀 공격 (낮은 확률) | 연금술 솥 독 물약 재료 |
| 생고기 | 몬스터 드롭 | 야영지 요리 |
| 빈병 | **수풀** 공격 (확률) / 포션 사용 후 | 연금술 솥 물약 제작 |
| 나무 | **항아리** 파괴 (확률) / 상자 | 야영지 점화 |
| 횃불 | 상자/제작 | 시야 확장 + 야영지 점화 |

> **수풀 드롭 테이블**: 나뭇가지(30%) / 약초(25%) / 돌(20%) / 빈병(15%) / 물약(7%) / 주문서(3%)
> **항아리 결과**: 골드·아이템 드롭(40%) / 빈 항아리(30%) / 몬스터 소환(20%) / 저주 상태이상(10%)

### 명명된 약초 10종 — 직접 섭취 vs 솥 제작

약초 이름을 알게 된 후에는 섭취 효과가 **고정**된다.  
직접 섭취 = 포션 효과의 약 30% (약하지만 즉시 사용 가능).  
솥에서 제작 = 포션 풀 효과.

| 약초 | 직접 섭취 효과 | → 솥 포션 |
|------|--------------|----------|
| 산삼 뿌리 | HP +max_hp×25% | 회복 물약 |
| 말린 영지버섯 | 2턴 HP 재생 | 재생 물약 |
| 만드라고라 뿌리 | MP +max_mp×25% | 정신력 물약 |
| 나이트쉐이드 잎 | 독 상태이상 (자신) | 독 물약 |
| 화염초 꽃잎 | 화염 상태이상 (자신) | 화염 물약 |
| 생마늘 | 상태이상 1개 해제 (우선순위: 독>화염>마비>슬로우) | 정화 물약 |
| 꿈결초 꽃잎 | 수면 1턴 + 피로도 -30 | 수면 물약 |
| 말린 피이끼 | 마비 1턴 (자신!) | 마비 물약 |
| 식은 얼음송이 | 느려짐 1턴 (자신!) | 얼음 물약 |
| 암브로시아 꽃 | 경험치 +50 | 경험치 물약 |

## 축복 & 저주

모든 아이템은 확률적으로 축복/저주 상태로 드롭된다.

```
축복 아이템  → 성능 한 단계 강화 (장비), 효과 증폭 (소모품)
저주 아이템  → 성능 한 단계 하향 + 장착 해제 불가 (장비)
저주 해제   → 정화 물약 사용 / 상인에서 저주 해제 구매

저주 장비 팁:
  힘 요건 미충족 상태에서 미확인 장비 착용 시
  저주이면 해제 불가 → 힘 충족 장비만 착용할 것
```

## 보물상자

```
층당 최소 1개, 최대 2개 보장
```

## HUD 하단바 레이아웃

```
[대기] [층수] │ [무기칸] [방패칸] [갑옷칸] [가방] │ (공간) │ [스킬1] [제작]
```

- **장착 칸 3개** (무기/방패/갑옷): 장착 중인 장비를 상시 표시. 탭하면 가방 팝업 오픈
- **가방 버튼**: 탭하면 가방 팝업 오픈

### 가방 팝업 동작

- 식량·물약 사용 후에는 가방이 **닫히지 않음** (액션 팝업만 닫힘, 가방 유지)
- 장착·던지기·버리기 등 다른 액션은 기존대로 가방 닫힘

탭 시 전체 화면 팝업:

```
┌─────────── 가방 ────────────┐
│ ◆ 장착 중                  │
│  [무기]  [방패]  [갑옷]     │  ← 탭 → 해제 팝업
├────────────────────────────┤
│ ◆ 가방                     │
│  [  ][  ][  ][  ][  ]      │  ← 인벤 5×2 그리드
│  [  ][  ][  ][  ][  ]      │  ← 탭 → 아이템 액션 팝업
└────────────────────────────┘
```

- 장착 슬롯 탭 → "해제" 팝업 (인벤이 꽉 찼으면 불가)
- 인벤 아이템 탭 → 기존 액션 팝업

## 아이템 액션

가방 팝업에서 아이템 탭 → 액션 팝업

| 액션 | 설명 |
|------|------|
| 사용 | 효과 발동, 인벤에서 제거 |
| 장착 | 장비 슬롯에 장착 (장비 아이템만) |
| 던지기 | 던지기 타겟팅 모드 진입 |
| 버리기 | 아이템 제거 |

### 던지기 타겟팅
```
범위: 5칸 이내 이동 가능한 타일 하이라이트
탭으로 투척, 대기 버튼 또는 취소 버튼으로 취소
```
