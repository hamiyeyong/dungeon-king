## 개요

인벤토리에서 재료를 선택해 아이템을 제작하거나 장비를 분해한다.
**물약 제작은 맵의 연금술 솥 오브젝트에서만 가능하다.**

## 관련 씬/노드

`HUD` · `Player` · `Map`

## UI 접근

HUD 우측 하단 **[제작]** 버튼 → 제작 팝업 (3탭)

```
┌──────────────────────────────────┐
│              제  작              │
│  [장비]      [물약]      [기타]  │  ← 탭 버튼
├──────────────────────────────────┤
│  목검(ATK+2) ← 나뭇가지×2       │  ← 장비 탭
│  나무 방패(DEF+1) ← 나뭇가지×3  │
│  ...                             │
│         영역 밖 터치로 닫기      │
└──────────────────────────────────┘
```

- **장비 탭**: 무기/방패/갑옷 레시피 — ATK/DEF 수치 함께 표시, 재료 충족 시 제작 가능
- **물약 탭**: 약초→물약 레시피 참고표 (솥에서만 제작 가능, 비활성)
- **기타 탭**: 횃불·강화 두루마리·수리도구

---

## 레시피 — 인벤에서 언제든 가능

티어 1 장비만 제작 가능. 고티어는 상자/상점/제단에서 드롭.

### 무기 (티어 1)
| 재료 | 결과 | ATK |
|------|------|-----|
| 나뭇가지 × 2 | 단검 | 1~10 |
| 나뭇가지 + 돌 | 숏소드 | 1~11 |
| 돌 × 2 | 곤봉 | 1~10 |
| 나뭇가지 × 3 | 벌목도끼 | 2~13 |
| 나뭇가지 × 1 | 지팡이 | 1~5 |
| 나뭇가지 × 2 + 천 × 1 | 숏보우 | 1~10 |

### 방어구 (티어 1~2)
| 재료 | 결과 | DEF |
|------|------|-----|
| 천 × 2 | 천 옷 | +2 |
| 천 + 돌 | 가죽 갑옷 | +4 |
| 나뭇가지 × 2 | 나무 방패 | +4 |

### 도구
| 재료 | 결과 | 효과 |
|------|------|------|
| 나뭇가지 + 천 | 횃불 | 시야 반경 확장 (보유 시) |
| 약초 × 2 + 나뭇가지 × 1 | 강화 두루마리 | 무기 +1강 + 내구도 완전 회복 |
| 광석 × 1 | 수리도구 | 장비 내구도 절반 회복 + 최대 내구도 -1 |

## 연금술 솥 — 맵 오브젝트 전용 (물약 제작)

**인벤에서 직접 물약 제작 불가. 반드시 연금술 솥 오브젝트에서만 제작.**

솥에 접근 → 인벤의 약초류 목록이 팝업으로 표시 → 재료 1종 선택 → 빈병 ×1 + 선택 재료 ×1 소모 → 물약 생성

```
흰 솥 (WHITE_CAULDRON)  → 랜덤일 경우 좋은 물약 (회복/포만/정화) 기준
검은 솥 (BLACK_CAULDRON) → 랜덤일 경우 나쁜 물약 (독/화염/수면) 기준
```

### 물약 레시피 (약초 재료 → 결과)
| 재료 | 결과 물약 | 비고 |
|------|----------|------|
| 약초 (일반) | 랜덤 물약 | 솥 종류에 따라 좋은/나쁜 물약 |
| 말린 피이끼 | 회복 물약 | 이름 공개 (식별됨) |
| 산삼 뿌리 | 포만 물약 | 이름 공개 |
| 암브로시아 꽃 | 회복 물약 | 이름 공개 |
| 말린 영지버섯 | 포만 물약 | 이름 공개 |
| 나이트쉐이드 잎 | 독 물약 | 이름 공개 |
| 화염초 꽃잎 | 화염 물약 | 이름 공개 |
| 만드라고라 뿌리 | 수면 물약 | 이름 공개 |
| 꿈결초 꽃잎 | 수면 물약 | 이름 공개 |
| 식은 얼음송이 | 수면 물약 | 이름 공개 |
| 생마늘 | 정화 물약 | 이름 공개 |

> 특정 약초 재료로 제작 시 해당 물약 타입이 **즉시 식별**된다 (이름 공개 상태로 획득).

## 야영지 전용 레시피 (요리)

야영지에서 불 점화 후에만 가능.

| 재료 | 결과 | 효과 |
|------|------|------|
| 생고기 | 구운 고기 | 배고픔 -50 |
| 약초 | 약초차 | 피로도 -30 |

### 야영지 점화 조건
```
나무 + 약초  →  불 점화
횃불        →  불 점화
```

---

## 분해

```
장비 아이템 선택 → [분해] → 재료 반환 (손실 있음)

무기 분해: 광석 1~2개 반환
방어구 분해: 천 or 광석 1~2개 반환
```

> 분해가 광석 획득의 주요 경로 — 수리도구 제작에 필수.

---

## 빈병 시스템

```
포션 음용 → 50% 확률로 빈병 1개 인벤에 추가
빈병은 연금술 솥에서 물약 제작 시 필수 재료 (약초류 ×1 + 빈병 ×1)
```
