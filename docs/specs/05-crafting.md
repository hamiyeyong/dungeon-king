## 개요

인벤토리에서 재료를 선택해 아이템을 제작하거나 장비를 분해한다.
**물약 제작은 맵의 연금술 솥 오브젝트에서만 가능하다.**

## 관련 씬/노드

`HUD` · `Player` · `Map`

## UI 접근

HUD 우측 하단 **[제작/분해]** 버튼 → 제작 팝업

```
┌─────────────────────────────┐
│  제작                        │
│  재료: [나뭇가지] [돌]  → 돌창│
│  재료: [천] [천]       → 천갑옷│
│  ...                        │
│  [닫기]                     │
└─────────────────────────────┘
```

## 레시피 — 인벤에서 언제든 가능

### 무기
| 재료 | 결과 | 효과 |
|------|------|------|
| 나뭇가지 × 2 | 목검 | ATK +2 |
| 나뭇가지 + 돌 | 돌창 | ATK +3 |
| 돌 + 돌 | 돌도끼 | ATK +5 |

### 방어구
| 재료 | 결과 | 효과 |
|------|------|------|
| 천 + 천 | 천 갑옷 | DEF +2 |
| 천 + 돌 | 가죽 갑옷 | DEF +4 |

### 도구
| 재료 | 결과 | 효과 |
|------|------|------|
| 나뭇가지 + 천 | 횃불 | 시야 반경 +3 (보유 시) |
| 약초 × 2 + 나뭇가지 × 1 | 강화 두루마리 | 무기 +1강 + 내구도 완전 회복 |
| 광석 × 1 | 수리도구 | 장비 내구도 절반 회복 + 최대 내구도 -1 |

## 연금술 솥 — 맵 오브젝트 전용 (물약 제작)

**인벤에서 직접 물약 제작 불가. 반드시 연금술 솥 오브젝트에서만 제작.**

```
흰 솥 (WHITE_CAULDRON)  → 빈병 + 재료 2개 → 물약 (발현도 높은 재료 기준)
검은 솥 (BLACK_CAULDRON) → 빈병 + 재료 2개 → 물약 (발현도 낮은 재료 기준)
```

### 물약 레시피 (재료 조합 → 효과)
| 주재료 | 보조재료 | 결과 물약 |
|--------|---------|----------|
| 약초 | 약초 | 회복 물약 |
| 독초 | 약초 | 독 물약 |
| 독초 | 독초 | 강독 물약 |
| 약초 | 빈병 (보조 생략) | 포만 물약 |

> 발현도: 재료마다 발현 효과 수치가 있으며, 흰 솥은 높은 쪽, 검은 솥은 낮은 쪽 효과가 나온다.
> 구체적 발현도 테이블은 추후 확정.

## 야영지 전용 레시피 (요리)

야영지에서 불 점화 후에만 가능.

| 재료 | 결과 | 효과 |
|------|------|------|
| 생고기 | 구운 고기 | 배고픔 -50 |
| 약초 | 약초차 | 피로도 -30 |

### 야영지 점화 조건
```
나무 + 약초  →  불 점화
횃불        →  불 점화
```

---

## 분해

```
장비 아이템 선택 → [분해] → 재료 반환 (손실 있음)

무기 분해: 광석 1~2개 반환
방어구 분해: 천 or 광석 1~2개 반환
```

> 분해가 광석 획득의 주요 경로 — 수리도구 제작에 필수.

---

## 빈병 시스템

```
포션 사용 → 빈병 1개 인벤에 자동 추가
빈병은 연금술 솥에서 물약 제작 시 필수 재료
```
