# Spec: 마법 시스템 (Magic System)

> Status: 기획 중
> Priority: Phase 2

---

## 개요

주문서(1회용 아이템)와 별개로, **고대 주문서**를 통해 마법을 영구 학습(메모라이즈)하면
MP 소모로 반복 사용할 수 있다.
마법사가 MP 효율이 높지만, 모든 직업이 사용 가능.

## 관련 씬/노드

`Player` · `HUD`

## 주문서 vs 고대 주문서

| 구분 | (일반) 주문서 | 고대 주문서 |
|------|--------------|-----------|
| 사용 | 인벤에서 사용 → 효과 즉시 발동 → 소멸 | 인벤에서 사용 → 마법 영구 학습 (소멸) |
| 반복 사용 | 불가 (1회용) | 가능 (MP 소모) |
| 발동 시 정보 | 사용 시 정체 공개 | 학습 시 정체 공개 |
| 취급 | 현재 구현된 BASH/TELEPORT/IDENTIFY | 추가 구현 대상 |

> 고대 주문서는 배우기만 하면 정신력이 허락하는 한 영구 사용 → 함부로 쓰지 말고 학습
> 불필요한 마법은 상인에게 파는 것이 낫다 (어차피 마법 강화 시 원하지 않는 마법이 강화될 수 있음)

## 무기 스킬

무기 종류마다 고유 스킬이 있다. MP 소모 25, 쿨다운 3턴(60 AP).

| 무기 종류 | 스킬 효과 |
|---------|---------|
| 격투무기 | 4연타 (평타 판정 — 룬·인챈트 전부 적용) |
| 지팡이 | 4연타 (빠른 공격 속도와 시너지) |
| 장검 | 3연타 |
| 양손무기 | 주변 3×3 2연타 AOE |
| 단검 | 양손 내리찍기 (이도류 시 데미지 2배) |
| 둔기 | 단타 + 기절 (후속타는 기절 판정 없음) |
| 활 | 다단 원거리 (원거리 판정) |

## 마법 분류

### 공격 마법

| 마법 이름 | 효과 | 학습 MP 소모 | 비고 |
|-----------|------|------------|------|
| 매직 미사일 | 원거리 단일 공격 | 낮음 | 마법사 패시브 기본 보유. 초반 원거리 수단. 인챈트 시너지 |
| 화염구 | 원거리 폭발 + 범위 피해 + 불붙음 | 높음 | 바리케이드·문 파괴 가능 |
| 정신의 폭풍 | 광역 정신력 공격 | 높음 | 다수의 적 처리 |
| 얼음 회오리 | 넓은 범위 피해 + 빙결 | 높음 | 전사 직업 스킬과 동일 효과. 물 타일에서 더 강함 |
| 부패의 구름 | AOE 독가스 | 중간 | 지속 피해 |
| 자연의 번개 | 단일 + 확률 기절 | 중간 | 물 타일에서 광역으로 변함. 범용성 최고 |
| 독 안개 | 자신 주변 AOE 독 | 중간 | 신체 마법 분류 |
| 사자의 포효 | 광역 공격 | 높음 | 정신의 폭풍과 조합 |
| 사신 소환 | 강력한 소환 + HP 지속 소모 | 매우 높음 | 체력 담보 → 흡혈 무기 시너지 |

### 방어/회복 마법

| 마법 이름 | 효과 | 비고 |
|-----------|------|------|
| 재생 | HP 서서히 회복 (여러 턴) | 기본 회복 수단 |
| 나무껍질 갑옷 | 방어막 (최대 HP의 일정 %) | 초반 물리 공격 대상 거의 무적. 후반 마법 공격엔 약함 |
| 신성한 빛 | 공격 + 실명 부여 | 약간 수비적 |
| 차가운 손길 | 공격 + 느림 부여 | 생존력 향상 |

### 유틸 마법

| 마법 이름 | 효과 | 비고 |
|-----------|------|------|
| 투명화 | N턴간 적에게 보이지 않음 | 기습 연계, 위기 탈출 |
| 심안 | 층 전체 지도 표시 | 숨겨진 방, 함정 포함 가능 |
| 마법지도 | 층 전체 함정·아이템 위치 표시 | |
| 순간이동 | 시야 내 타일로 이동 (마법 판정) | 현재 주문서로 구현된 것과 별개 |
| 투시 | 범위 내 벽 너머 투시 | 함정·비밀통로 발견 보조 |
| 디스펠 | 자신의 상태이상·저주 전부 해제 | 전 직업 필수 마법 |
| 간절한 기도 | 랜덤 회복/버프 | MP 소모 매우 높음, 의지하기 어려움 |
| 공중 부양 | 절벽 위 이동 가능 | 탐색용 |
| 윈드워크 | 모든 함정 무시 이동 | |

## 마법 학습 방법

```
고대 주문서 인벤에서 사용 → 마법 1개 영구 학습 (주문서 소멸)
마법사의 석상 + 영웅의 돌 → 보유 마법 중 랜덤 3개 선택지 → 1개 강화
                              (보유 마법 없으면 랜덤 1개 신규 학습)
```

- 학습 시 정체 공개
- 취소해도 주문서는 돌려주지 않음
- **쓸모없는 마법이면 마법 강화 대신 적에게 던져서 확인 후 상인에 파는 것이 이득**

---

## 마법 강화

- 레벨 1~5까지 강화 가능
- 강화 방법: 마법 강화 주문서, 마법사의 석상 + 영웅의 돌
- 강화 시 데미지·범위·지속시간 증가 / MP 소모 감소
- 너무 많은 마법을 배우면 원치 않는 마법이 강화될 수 있음 → 주력 마법 2~3개 집중

---

## 마법 발동

- **MP 부족 시**: HP를 소모해서 발동 가능
- 마법사 패시브(마법 친화): 경갑 착용 시 MP 자연회복 속도 향상

---

## HUD 마법 슬롯

현재 스킬 슬롯 2개 → 마법 슬롯으로 확장:
- 슬롯 1/2: 학습한 마법 중 선택해 단축키 배치
- 인벤 내 주문서와 별개로 표시
- 발동 시 MP 소모, 재사용 가능

---

## 특수 상호작용

```
- 화염 계열 마법 → 수풀/바리케이드/문 불붙임 (연소 전파 주의)
- 자연의 번개   → 물 타일에서 광역으로 변환
- 나무껍질 갑옷 → 씨앗 대신으로도 활용 가능 (시너지: 사냥꾼 룬 신비의 씨앗)
- 사신 소환     → 소환 시 HP 지속 소모 → 흡혈 속성 무기로 커버
```
