# Spec: 스탯 & 게이지 시스템 (Stats & Gauges)

> Status: 브레인스토밍 완료, 구현 대기

## 개요

HP / MP / 배고픔 / 피로도 4개 게이지.
배고픔·피로도는 수치로 관리하며 임계치 초과 시 상태이상으로 전환.

## 관련 씬/노드

`Player` · `HUD`

## 의존 스펙

- [[02-monster-combat]] — 전투 소모 (피로도 +3/회), 상태이상 데미지
- [[03-item-system]] — 음식/포션/야영지 회복 수치

## 게이지 정의

| 게이지 | 표시 | 소모 | 회복 |
|--------|------|------|------|
| **HP** | HUD 바 | 전투, 상태이상, 굶주림 | 포션, 야영지 휴식 |
| **MP (정신력)** | HUD 바 | 주문서 발동 | 자연회복 (피로도 낮을 때) |
| **배고픔** | 수치 (아이템창) | 이동 1칸당 증가 | 음식 섭취 |
| **피로도** | 수치 (아이템창) | 전투 1회당 증가 (이동보다 많음) | 야영지 휴식, 대기(소량) |

---

## 배고픔 임계치

```
정상 (0~449)      → 아무 영향 없음
배고픔 (450~599)  → HP 자연회복 중단
굶주림 (600)      → 일정 턴마다 HP -1 감소
```

- 회복: 음식 섭취 (신선한 음식 배고픔 -30, 구운 고기 -50)
- 이동 1칸 = 배고픔 +1

---

## 피로도 임계치

```
정상 (0~449)      → 아무 영향 없음
피곤함 (450~599)  → MP 자연회복 중단
탈진 (600)        → 최대 HP가 점점 감소 (해제 시 서서히 회복)
```

- 회복: 야영지 휴식 (피로도 초기화), 대기 1턴 (-3)
- 전투 1회 = 피로도 +3
- 이동 1칸 = 피로도 +0 (전투만 소모)

---

## MP (정신력)

- 주문서 발동 시 MP 소모
- MP 부족 시: HP를 소모해서 발동 가능 (던죽싫 방식)
- 피로도 낮을 때 턴마다 소량 자연회복
- 피곤함/탈진 상태: 자연회복 중단

### 주문서별 MP 소모 (예시, 미확정)
```
강타 주문서      MP -15
독안개 주문서    MP -20
순간이동 주문서  MP -25
투명화 주문서    MP -20
식별 주문서      MP -10
```
