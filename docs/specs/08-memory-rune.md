# Spec: 기억의 상자 & 룬 시스템 (Memory Chest & Rune System)

> Status: 기획 중
> Priority: Phase 2

---

## 개요

런이 끝날 때(사망 또는 보스 클리어) **기억의 상자**를 획득한다.
상자를 열면 **룬 파편**과 **고대의 주화**를 얻을 수 있다.
이 자원으로 **룬**을 해금·레벨업하여 다음 런부터 캐릭터에게 패시브 능력을 부여한다.

> 현재 07-home-exploration-xp.md의 탐험XP 마일스톤은 별개 시스템으로 유지한다.
> 기억의 상자는 룬/파편 경제 전용.

## 기억의 상자 (Memory Chest)

### 획득 조건

| 조건 | 상자 등급 |
|------|-----------|
| 캐릭터 사망 | 일반 |
| 5층 보스 클리어 | 희귀 |
| 10층 보스 클리어 | 에픽 |
| 게임 클리어 (25층+) | 레전드 |

### 상자 오픈 — 고대의 주화 소모

상자는 런 종료 시 자동 획득되며, **오픈할 때 고대의 주화를 소모**한다.

| 상자 등급 | 오픈 비용 | 파편 지급 |
|-----------|-----------|-----------|
| 일반 | 10~20 주화 | 1~3개 |
| 희귀 | 30~50 주화 | 3~6개 |
| 에픽 | 60~100 주화 | 6~10개 |
| 레전드 | 150~250 주화 | 15~25개 |

- 파편은 랜덤 룬 1~3종에 분배
- 아직 제작 안 된 룬의 파편도 쌓임 → 제작 임계치 도달 시 제작 가능 표시
- 주화가 부족하면 상자는 보관되고 나중에 주화가 생기면 오픈 가능

> 고대의 주화 획득처: 런 중 탐험 보상, 층 보스 처치, 고대 제단 등

### 홈 화면 연출

```
게임 종료 → 홈으로 돌아옴 → "기억의 상자 획득!" 팝업
→ [오픈 (주화 N개)] / [보관] 선택
→ 오픈 시: 열기 애니메이션 → 파편 N개 획득 메시지
→ 닫기 → 홈 화면으로
```

## 룬 시스템

### 슬롯 해금

런 시작 전 홈 화면에서 룬을 선택해 장착한다.
최고 층 기록에 따라 슬롯이 해금된다.

| 최고 기록 | 해금 슬롯 |
|-----------|----------|
| 없음 (첫 런) | 1 슬롯 |
| 5층 | 2 슬롯 |
| 10층 | 3 슬롯 |
| 15층 | 4 슬롯 |
| 20층 | 5 슬롯 (최대) |

### 룬 등급

| 등급 | 제작 파편 | 최대 레벨 | 강화 1레벨당 파편 (초기→후반) |
|------|-----------|-----------|-------------------------------|
| 일반 | 10개 | 60 | 2개 → 최대 10개 |
| 희귀 | 15개 | 35 | 3개 → 최대 15개 |
| 에픽 | 20개 | 12 | 5개 → 최대 20개 |
| 레전드 | 30개 | 5 | 10개 → 최대 30개 |

### 룬 제작 (해금)

```
파편 N개 누적 → [제작] 버튼 활성화 → 클릭 시 룬 제작 완료 (파편 소모)
제작된 룬은 Lv1 상태로 장착 가능
```

- 파편은 상자에서 랜덤 획득 (제작 전부터 쌓임)
- **주화 불필요** — 파편만으로 제작

### 룬 강화 (레벨업)

```
제작된 룬 + 파편 N개 → [강화] → 룬 레벨 +1
레벨이 높아질수록 필요 파편 수 증가
```

- **주화 불필요** — 파편만으로 강화
- 강화 비용 공식: `base_cost + floor(level / 5)` (등급별 base_cost 상이)

### 고대의 주화 사용처

고대의 주화는 **룬 강화에 쓰지 않는다**. 아래 두 용도로만 사용:

1. **기억의 상자 오픈** — 런 종료 후 획득한 상자를 오픈할 때 주화 소모
2. **고대 상점** — 주화로 룬 상자를 직접 구매 → **즉시 오픈** (보관 없음)

## 고대 상점 (Ancient Shop)

> 현재는 룬 상자만 판매. 추후 다른 상품 확장 예정.

### 상자 종류

| 상자 종류 | 주화 비용 | 내용 |
|-----------|-----------|------|
| 랜덤 룬 상자 | 20 | 등급 랜덤 결정 후 파편 지급 |
| 일반 룬 상자 | 15 | 파편 1~3개 |
| 희귀 룬 상자 | 35 | 파편 3~6개 |
| 에픽 룬 상자 | 80 | 파편 6~10개 |
| 레전드 룬 상자 | 200 | 파편 15~25개 |

### 랜덤 상자 등급 확률

| 등급 | 확률 |
|------|------|
| 일반 | 55% |
| 희귀 | 30% |
| 에픽 | 12% |
| 레전드 | 3% |

### 상점 구매 흐름

```
[구매] 클릭 → 주화 차감 → 상자 오픈 연출

랜덤 상자: "???" 애니메이션 → 등급 공개 → 파편 N개 획득
등급별 상자: 바로 상자 오픈 연출 → 파편 N개 획득
```

## 저장 구조

```
save_data.cfg [meta] 섹션에 추가:

[runes]
unlocked=["질긴_생명력","전투_감각"]  # 해금 룬 id 목록
levels={"질긴_생명력":3,"전투_감각":1}
fragments={"질긴_생명력":8,"무기_방어술":2}
equipped=["질긴_생명력","전투_감각"]  # 장착 순서 (슬롯 순)
coins=150  # 고대의 주화 보유량
```

## 저장 구조 변경 (v2)

```
[runes]
fragments={"질긴_생명력":3,"무기_방어술":12}  # 미제작 포함 전체 파편 누적
crafted=["질긴_생명력","전투_감각"]           # 제작 완료된 룬 id
levels={"질긴_생명력":3,"전투_감각":1}        # 제작 룬의 현재 레벨
equipped=["질긴_생명력","전투_감각"]          # 장착 순서 (슬롯 순)
coins=150                                    # 고대의 주화
pending_chests=["normal","rare"]             # 미오픈 보관 상자
```

> 기존 `unlocked` 키는 `crafted`로 rename. 저장 데이터 마이그레이션 필요.
