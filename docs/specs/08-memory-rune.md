# Spec: 기억의 상자 & 룬 시스템 (Memory Chest & Rune System)

> Status: 기획 중
> Priority: Phase 2

---

## 개요

런이 끝날 때(사망 또는 보스 클리어) **기억의 상자**를 획득한다.
상자를 열면 **룬 파편**과 **고대의 주화**를 얻을 수 있다.
이 자원으로 **룬**을 해금·레벨업하여 다음 런부터 캐릭터에게 패시브 능력을 부여한다.

> 현재 07-home-exploration-xp.md의 탐험XP 마일스톤은 별개 시스템으로 유지한다.
> 기억의 상자는 룬/파편 경제 전용.

## 기억의 상자 (Memory Chest)

### 획득 조건

| 조건 | 상자 등급 | 파편 | 주화 |
|------|-----------|------|------|
| 캐릭터 사망 | 일반 | 1~3개 | 10~20 |
| 5층 보스 클리어 | 희귀 | 3~6개 | 30~50 |
| 10층 보스 클리어 | 에픽 | 6~10개 | 60~100 |
| 게임 클리어 (25층+) | 레전드 | 15~25개 | 150~250 |

- 파편은 **보유한 룬 중 1~3종** 랜덤 지급
- 아직 해금 안 된 룬의 파편도 드롭될 수 있음 → 해금 임계치 도달 시 룬 해금

### 홈 화면 연출

```
게임 종료 → 홈으로 돌아옴 → "기억의 상자 획득!" 팝업
→ 열기 애니메이션 → 파편 N개 + 주화 M개 획득 메시지
→ 닫기 → 홈 화면으로
```

## 룬 시스템

### 슬롯 해금

런 시작 전 홈 화면에서 룬을 선택해 장착한다.
최고 층 기록에 따라 슬롯이 해금된다.

| 최고 기록 | 해금 슬롯 |
|-----------|----------|
| 없음 (첫 런) | 1 슬롯 |
| 5층 | 2 슬롯 |
| 10층 | 3 슬롯 |
| 15층 | 4 슬롯 |
| 20층 | 5 슬롯 (최대) |

### 룬 등급

| 등급 | 최대 레벨 | 획득 난이도 |
|------|-----------|------------|
| 일반 | 60 | 높음 |
| 희귀 | 35 | 보통 |
| 에픽 | 12 | 낮음 |
| 레전드 | 5 | 매우 낮음 |

### 레벨업

```
파편 N개 + 고대의 주화 M개 → 해당 룬 레벨 +1
레벨이 높아질수록 필요 파편/주화 수 증가
```

## 저장 구조

```
save_data.cfg [meta] 섹션에 추가:

[runes]
unlocked=["질긴_생명력","전투_감각"]  # 해금 룬 id 목록
levels={"질긴_생명력":3,"전투_감각":1}
fragments={"질긴_생명력":8,"무기_방어술":2}
equipped=["질긴_생명력","전투_감각"]  # 장착 순서 (슬롯 순)
coins=150  # 고대의 주화 보유량
```
