## 스탯 시스템

### 스탯 종류 및 효과

| 스탯 | 효과 |
|------|------|
| **힘 (STR)** | 장비 착용 제한, ATK 보정. 힘 20 달성 시 방패 방어력 2배 |
| **민첩 (AGI)** | 회피율 보정. 도적 패시브: 경갑 착용 시 민첩 수치 = 회피 보너스 % |
| **체력 (VIT)** | 최대 HP 보정 |
| **지능 (INT)** | 최대 MP 보정, 마법 데미지 |
| **운 (LUK)** | 크리티컬 확률, 아이템 드롭률 |

### 장비 착용 제한 (힘)

힘 수치보다 무거운 장비 착용 시:
- 명중률 하락
- 회피율 하락
- 행동 속도(AP) 증가 → 턴당 손해

> 저주받은 고티어 장비를 착용하면 힘 부족 상태에서 해제 불가 → 굶어 죽을 위험

## 직업별 액티브 스킬

**공통**: 직업 스킬 MP 소모 35, 쿨다운 5턴(100AP)

| 직업 | 스킬 이름 | 효과 |
|------|-----------|------|
| 전사 | 지면 강타 | 인접 3×3 범위 2연타 AOE — 넓은 범위, 단 적이 뭉쳐야 효율적 |
| 마법사 | 얼음 회오리 | 넓은 범위 피해 + 빙결. 무기 공격력에 비례. 물 타일 위의 적에게 추가 피해 |
| 도적 | 섬광탄 | 화면 전체 피해 + 적 실명. 광역기라 다수의 적에게 매우 강력 |
| 사냥꾼 | 난사 | 현재 원거리/투척 무기로 3~6연타. 평타 판정이므로 룬·인챈트 전부 적용. 단일 대상에게 매우 강력 |

> 사냥꾼의 난사: 원거리 무기 없이 투척 무기로도 발동. 스턴 다트 5개로 난사 → 보스 마비 6연타.

## AP 시스템

- **1턴 = 20 AP**
- 이동 1칸 = 10 AP
- 공방 1회 교환 = 20 AP (내 공격 10 + 적 공격 10)
- 무기 스킬 쿨다운 = 3턴 = 60 AP → 이동 6칸 or 공방 3회 후 재사용
- 직업 스킬 쿨다운 = 5턴 = 100 AP
- 격투무기/지팡이: 1번 공격 = 0.5턴(10 AP) → 평타 후 스킬 사용 또는 한 대 때리고 후퇴 가능

## 회피율 메커니즘

- 개활지(주변 빈 타일 많음) → 회피율 높음
- 문턱에 적을 끼운 상태로 문 밖에서 공격 → 최적 포지션 (회피 유지 + 다굴 방지)
- 특히 도적: 경갑 착용 시 민첩 수치 그대로 회피 보너스 %로 추가

## 직업별 육성 방향 (공략 요약)

### 전사
- 힘/체력 높아 미확인 장비도 부담 없이 착용 가능
- 중갑 + 방패로 안정적 방어 OR 양손 무기로 화력 집중
- 힘 물약 우선, 고티어 무기 확보 후 방패/쳐내기 세팅 추천
- 룬에 의존도 낮음 → 룬 슬롯 여유로움

### 마법사
- 지팡이 공격속도 빠름(0.5턴) → 초반 평타+무기스킬 콤보로 버팀
- 힘 물약 우선 → 장비 착용 위해 / 이후 지능 물약
- 저주에 매우 취약: 힘↓ 지팡이 데미지↓, 지능↓ MP 관리 불가, 체력↓ 개복치
- 주력 마법: 자연의 번개(물 타일 시너지), 나무껍질 갑옷
- 격투무기 + 중갑 조합 추천 (경갑 패시브 효과 있지만 생존이 우선)

### 도적
- 경갑 강제 착용 (패시브 효과 때문)
- 개활지 전투 필수 — 회피율 극대화
- 기습(잠든 적 or 그림자 반지) → 크리티컬 → 섬광탄 연계
- 살인적인 기습 룬 확보 시 완전체
- 힘/민첩 올라가는 것 우선으로 물약 제작
- 단검 → 쌍단검 → 크리티컬 빌드

### 사냥꾼
- 룬 의존도 가장 높음 — 살인적인 기습 + 저격 조합 필수
- 체력 낮아 초반이 가장 어려움
- 스턴 다트 난사로 보스 처리
- 고티어 활 + 미스릴 화살 = 핵폭탄 화력
- 화살 보급: 패시브 처치 드롭 + 나무/강철로 자체 제작
