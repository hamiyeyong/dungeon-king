## 개요

- 지능·레벨이 높을수록 위력 강화
- 고대 주문서로 동일 마법 재습득 시 +1 강화 (소모 정신력 불변, 위력 증가)
- 시전 정신력 부족 시 HP 대신 소모 가능
- 적성 기준 소모 정신력: 낮음 ≈ 26 / 보통 ≈ 39 / 높음 ≈ 52 (적성 상이면 감소)
- 직업별 적성 상: 전사 → 신체 / 마법사 → 아케인 / 사냥꾼 → 자연 / 도적 → 없음(중 多)
- 신성 마법은 전 직업 적성 하

**강화 우선순위**: 자연의 번개 · 사자의 포효 · 치유처럼 강화 효율 좋은 것을 3개 이하로만 배우고 집중 강화. 강화 불가 주문(신성한 기도 등)은 고민 없이 습득.

| 마법 | 효과 |
|------|------|
