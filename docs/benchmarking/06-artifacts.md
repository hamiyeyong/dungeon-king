# 아티팩트

[← 로드맵](README.md)

---

## 시스템

이상한 우물에 장비를 던지면 아티팩트화 (고티어 권장). 강화·인챈트 계승 + 고정 인챈트 1개.  
레시피 레벨 상승 시 티어 보너스 (희귀 +0.04/레벨, 에픽 +0.12, 레전드 +0.30).  
**장신구 아티팩트**: 장신구 3개 + 마력 조각 2~4개 + 랜덤 재료 1개. 고정 인챈트 1 + 랜덤 인챈트 0~3.

| 수호의 허리띠 | 에픽 | 수호의 (데미지 흡수 방어막) |

| 속도의 부츠 | 에픽 | 속도의 (이동속도·회피율↑) |

## 주요 빌드 세트

| 빌드 | 핵심 아티팩트 |
|------|-------------|
| 방패 전사 | 왕의 타바르 + 철벽의 갑옷 + 고위 성기사의 방패 (방패 방어율 80~90%) |
| 기습 도적 | 사신의 낫 + 바다의 무법자 + 살인적인 기습 룬 |
| 회피 도적 | 영원의 비수 + 약탈자의 갑옷 + 속도의 부츠 |
| 마법 법사 | 엘프의 유물 + 파괴의 로브 + 파괴의 브레이서 |
| 흡혈 법사 | 심장을 노리는 자 + 아무 방어구 (마법으로 원거리 공격 → 흡혈 버그 활용) |
