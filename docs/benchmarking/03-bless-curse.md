# 축복 & 저주 시스템

[← 로드맵](README.md)

---

## 개요

모든 아이템은 확률적으로 **축복** 또는 **저주** 상태로 등장.  
캐릭터가 스스로 저주를 걸 수는 없다.

---

## 축복 획득 방법

| 방법 | 확률 |
|------|------|
| 이상한 우물에 아이템 던지기 | 극악 (매우 낮음) |
| 천사 석상 — "커다란 축복을 받았다" 메시지 | 희귀. HP·MP 회복 / 스탯 증가 / 랜덤 아이템 축복 중 발동 |

---

## 축복 효과

| 분류 | 일반 효과 → 축복 추가/변화 |
|------|--------------------------|
| 장비 | 1티어 높은 것으로 취급. 5티어 축복 = 게임 최강. 최대 강화 시 스펙 차이 극대화 |
| 음식 | 배고픔 회복 +50%. 일정 확률로 랜덤 스탯 상승 |
| 물약 (부정 효과 계열) | 마실 때 완전히 다른 긍정 효과로 변환 |
| 물약 (회복·정화 계열) | 기존 효과 + 추가 효과 병행 |
| 스탯 물약 | 스탯 +2 (일반 +1) |
| 주문서 | 1단계 강화된 성능 + 정신력 소량 회복 |
| 장비 강화 주문서 | 장비 +2 강화 + 요구 힘 -2. (마법강화는 +1만, 대신 정신력 회복) |
| 고대 주문서 | 1단계 강화된 마법 습득. 미배운 상태 사용 시 정신력만 회복 |
| 씨앗 | 섭취 시 부작용 확률 대폭 감소. 심을 때는 차이 없음 |
| 소모성 아이템 공통 | 매우 낮은 확률로 **축복 버프** (행운 +2) |

## 저주 효과

| 분류 | 페널티 |
|------|--------|
| 장비 | 1티어 낮은 취급. **착용 해제 불가** (저주 해제 필요). 요구 힘은 기존과 동일 |
| 장신구 | 특수 효과 비정상 발동 |
| 음식 | 낮은 확률로 저주 상태이상 (스탯 1 감소). 허기 회복량은 일반과 동일 → 급할 때 먹어도 무방 |
| 물약 | 기존 효과 + 마실 시 저주 상태이상 (랜덤 스탯 1 감소). **던질 시 저주 없음** |
| 주문서 | 높은 확률 발동 실패 (턴 소모). 운 나쁘면 캐릭터도 저주 상태이상 |
| 고대 주문서 | 배우기 일정 확률 실패 |
| 씨앗 | 섭취 시 일정 확률 저주 상태이상. 심을 때는 차이 없음 |

> **저주 장비 팁**: 힘이 모자란 상태로 미확인 장비를 끼면 저주 시 해제가 불가 → 힘이 충족된 장비만 끼울 것.

---

## 저주 해제 방법

| 방법 | 효과 |
|------|------|
| 정화 물약 | 저주 포함 상태이상 제거. 오염된 샘에 넣으면 맑은 샘으로 변환 |
| 맑은 샘물 | HP·MP 회복 + 모든 상태이상 해제 (저주 포함) |
| 정화 마법 | 상태이상 1개 제거 (화면 좌상단 기준) |
| 디스펠 마법 | 모든 자신 상태이상 제거 + 아이템 저주 1개 해제 + 적 버프 제거 + 마법 봉인 |
| 저주 해제 주문서 | 아이템 1개의 저주 해제 |
