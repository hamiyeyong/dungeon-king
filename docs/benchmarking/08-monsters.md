## 공통 규칙

- ★★: 보스 / ★: 준보스. 전부 노말 기준 (하드모드는 전반적 스탯 상승)
- 모든 보스는 스킬 사용 직전 몸 전체가 붉게 번쩍 → 1턴 후 스킬 발동. 한 대 치고 물러나면 회피 가능
- 준보스는 넓은 방에서 등장
- 원거리 몬스터: 일직선 상이면 공격, 아니면 접근. 행동속도 대부분 느림
- 느린 근거리 몬스터: 먼저 다가가면 2연타 위험 → 제자리에서 유인

## 공통 몬스터

| 몬스터 | 특이사항 |
|--------|---------|
| 무너진 벽 해골 | 곡괭이·관에서 랜덤 소환. 일반 해골과 동일 |
| 유령 | HP 1. **일반 공격 거의 회피**. 무기/직업 스킬·마법·투척은 그대로 명중. 스턴 다트 투척이 최선 |
| 거대 유령 | 유령보다 HP 높고 회피율은 낮음. 큰 무덤에서만 출현 |
| 미믹 | 붉은 상자 = 미믹 가능성. 패치 후 체력·공격력 강화. Hard: 2칸 이동 |
| 움직이는 석상 | 넓은 방 중앙 등장. 들고 있는 무기 확정 드롭. 정보 확인 후 티어 보고 대응 |
| 미이라 | 관에서 낮은 확률 출현. 체력 높고 그 층 통상 몹보다 강함 |
| **★피라냐** | **4마리 팀. 최강 공격력. 절대 근접 금지.** 대응 방법: ① 자연의 번개(물타일에서 광역 2배) ② 투명화/공포의 속삭임으로 우회 ③ 공간 역전으로 육지 텔포 → 즉사 ④ 광란의 메아리로 내분 ⑤ 폭탄 (화염 아님, 물에서도 유효) ⑥ 직업+양손무기 스킬 |
