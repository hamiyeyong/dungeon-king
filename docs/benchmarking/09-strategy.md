## 직업 가이드

### 전사
**특징**: 기본 힘 최고, 무기 숙련 패시브로 어떤 무기든 무난. 초반 2티어 격투/단검 즉시 착용 가능. 체력 높아 중갑+방패 세팅에 특화. 초보자용.

**운영**:
- 힘 물약 올인
- 초반: 2티어 격투무기 또는 단검
- 후반: 둔기/한손검 + 방패 + 중갑
- 마법: 디스펠·시간 정지·공포의 속삭임 외에는 정신력 회복 수단으로만 활용
- 방패 세팅: 왕의 타바르 + 철벽의 갑옷 + 고위 성기사의 방패 → 방패 방어율 80~90%

### 마법사
**특징**: 지능 최고, 초반 매직 미사일로 원거리 커버. 체력·힘 낮아 미확인 템 착용·섭취 금물. 경갑 패시브(정신력 회복)가 있지만 생존을 위해 중갑 권장.

**운영**:
- 힘 제한 맞출 힘 물약 먼저, 이후 지능 물약
- 초반: 매직 미사일 + 경갑 + 근접 마무리
- 후반: 격투무기(마법 아티팩트) + 중갑 + 자연의 번개·나무껍질 갑옷
- 주력 마법: 자연의 번개 (물 타일 광역, 피라냐·악어·소환된 데몬 최강)
- 흡혈 법사 빌드: 심장을 노리는 자 + 원거리 마법 → 마법 사용 시에도 흡혈 발동 (버그 활용)

### 도적
**특징**: 높은 민첩+패시브로 회피 특화. 경갑 강제. 기습 시너지와 살인적인 기습 룬이 핵심.

**운영**:
- 힘/민첩 중 되는 것부터 올리기 (고정 순서 없음)
- 경갑 강제 착용. 중갑·플레이트는 보이는 족족 매각
- 반드시 개활지에서 싸우거나 문턱 1대1로 유인
- 핵심 룬: 살인적인 기습 + 치명적인 반격 (기습 데미지 극대화)
- 후반: 쌍단검 + 경갑 아티팩트 + 크리티컬 한 방 노리기

### 사냥꾼
**특징**: 체력 최하(9). 원거리 특화지만 화살 수급이 필수. 직업 스킬은 평타 판정 다단히트라 보스전 최강.

**운영**:
- 힘 물약 올인 (힘 제한이 가장 빡빡)
- 초반: 화살 기습 + 스턴 다트 난사 (5·10층 보스 쉽게 처리)
- 살인적인 기습 + 저격 룬 조합 갖추면 3티어 활로도 압도적 딜
- 미스릴 화살 1발 → 6발 복사 (사냥꾼 직업 스킬)로 보스전 화력 극대화

## 룬 조합

| 조합 | 효과 | 추천 직업 |
|------|------|----------|
| 살인적인 기습 + 저격 | 원거리 기습 극대화 | 사냥꾼 |
| 살인적인 기습 + 치명적인 반격 | 근거리 회피+기습 동시 | 도적 |
| 미식가 + 강한 소화력 + 마음의 양식 | 음식이 만능 회복약화 | 전사 |
| 머리강타 + 몰아치기 + 검술의 달인 | 격투 무기 2연타 무한 | 전사·도적 |
| 질긴 생명력 | 전체 체력 증가. 어느 직업에나 필수 | 전체 |

## 인 게임 핵심 팁

### 기본

- 격투/지팡이는 0.5턴 공격 → 1기본공격 + 스킬/아이템 콤보 가능
- **버그 콤보**: 1타 후 재접속 → 턴 초기화 (배고픔 소폭 상승). 최종보스에도 유효
- 투척 취소 버그: 투척/타겟 마법 버튼 눌렀다 취소 → 스킬 쿨타임 초기화 (이동하면 다시 돌아옴)
- 오브젝트 꾹 누르기 → 정보 확인. 문은 닫기 선택지도 포함
- 맵과 오브젝트 전부 조사. 서두르지 말 것 — 빨리 내려가면 그만큼 빨리 죽음

### 주문서

- 고대 주문서는 함부로 사용 금지 — 배우면 영구 사용. 미확인 시 사용하면 정신력만 회복
- 필요 없는 마법은 팔거나 즉시 사용 (효과 동일 + 정신력 회복)
- 미확인 주문서 확인법: 적 근처에서 사용 (원거리 무료 피해). 물 타일 위에서 확인 권장 (대지의 분노 불붙음을 즉시 해제)
- 장비 강화 주문서: 즉시 사용 말고 수리 후 강화 (내구도+힘 감소 동시)

### 타일 활용

- 흙 타일: HP·MP 자연회복 증가 (정보 보기 시 '바닥'으로 표시)
- 물·풀·아이템 위치에는 함정 없음
- 숨겨진 통로: 주로 횃불 주변 3칸 이내에서 발견
- 바리케이드·잠긴 문: 화염계 아이템으로 태울 수 있음 (열쇠 절약 비추 — 화염 아이템이 더 귀함)
- 가스 함정 (독·마비): 진행도에 따라 범위 증가 → 멀리서 던져 처리. 스파이크는 해제 불가

### 물약

- 미확인 아이템은 식별 전 상인 가격 1/3 — 의도적으로 8층까지 미식별 유지로 할인 가능
- 재생 물약 확인법: 상점에서 3개 이상 판매 시 거의 확정. 닫힌 문에 던질 수 있으면 화염 물약
- 물약 확인: 맑은 샘물에 던진 후 마시기 (능력치 물약도 가능)
- 독 물약 마실 때: 물 타일 위에서 마시면 독 빠르게 해제
- 검은 솥: 발현도 낮은 재료 기준 / 흰 솥: 발현도 높은 재료 기준

### 상점

- 3·8·13·18·23·28층 등장. 진입 순간 목록 확정
- 상점 내부에 있으면 적 진입·공격 불가 (원거리도) → 상점으로 유인 후 원거리 처치 가능
- 문턱에 서 있으면 공격받음
- 대지의 분노 주문서를 상점 근처에서 사용 금지 (주문서 소각)

## 배고픔 & 피로도

| 수치 | 효과 |
|------|------|
| 배고픔 450+ | 자연 체력 회복 중단 |
| 배고픔 600+ | 20턴마다 체력 감소 |
| 피로도 450+ | 정신력 자연 회복 중단 |
| 피로도 600+ | 최대 체력 점점 감소 |

- 배고픔은 1턴당 3 증가 / 피로도는 1턴당 1 증가
- 비상식량(400) vs 요리 고기(500+): 적시에 먹을 수 있으면 요리가 유리, 체력 급할 때는 비상식량이 낭비 없음
- 배고픔 18턴째에 음식 먹기 = 최대 절약 (19턴째는 먹는 행동에 1턴 소비로 데미지 입음)

## 야영지 활용

1. 그 층 탐색 완료 후 야영
2. **요리 먼저 (무한)** → 휴식 1회 (불 꺼짐, 피로도 초기화, HP·MP 회복)
3. 요리 재료: 날고기 + 약초 1개 + 마법 재료 1개
4. 야영 후 확률로 나쁜 이벤트 발생 (골드 도난, 재료 도난, 몬스터 등장)
   - **대응**: 고대 주문서·강화 주문서·마력 조각을 바닥에 내려놓고 잠 (착용 장비는 도난 없음)

## 장비 전략

### 아티팩트화 시점

- 4티어부터 올인 권장 (5티어는 드물어 기다리다 손해)
- 아티팩트 조합 전 인챈트 1회 + 조합 후 인챈트 1회로 총 2회 인챈트 가능 (강화 수치 계승)
- 내구도 0인 장비로 아티팩트 만들어도 새 장비 내구도에 영향 없음

### 강화 수칙

- 최대 강화 +10 (장신구 +5). **+9까지 강화 후 최종보스 전까지 유지** (최대 강화 후 수리 불가)
- 격투 무기: 내구도 소모 극히 빠름 → 광석 적극 수급
- 하드 모드: 25층부터 내구도 관리 필수

### 무기별 특성 요약

| 무기 | 장점 | 단점 | 추천 상황 |
|------|------|------|----------|
| 격투무기 | 0.5턴 공격, 룬·인챈트 발동 최다, 낮은 힘 제한 | 내구도 소모 빠름, 최대 데미지 낮음 | 중반, 룬 세팅 |
| 양손무기 | 광역 3×3 스킬, 높은 최대 데미지 | 방패 불가, 높은 힘 요구 | 포위전, 소환형 보스 |
| 검 | 무난함, 안정적 딜링 | 저강화 시 약함 | 범용 |
| 둔기 | 방패 세팅 아티팩트 多, 안정적 공격력 | 보조 무기 불가 | 전사 방패 세팅 |
| 단검 | 크리 확률 높음, 낮은 힘 제한 | 최대 데미지 낮음 | 도적 기습 |
| 활 | 원거리, 사냥꾼 패시브 화살 보충 | 화살 수급 의존, 높은 투자 필요 | 사냥꾼 전용 |

## 피라냐 처치법

1. **자연의 번개** — 물 타일에서 데미지 2배 + 2칸 광역. 4마리 동시 처리
2. **투명화/공포의 속삭임** — 싸우지 않고 지나치기. 주문서 2장 이상 필요
3. **공간 역전** — 육지로 텔포 → 즉사. 날고기를 주우러 돌아다녀야 하는 단점
4. **광란의 메아리** — 4마리 내분. 타락한 임프가 데몬을 잡아버리는 광경도 가능
5. **폭탄** — 화염 아님 = 물 위에서도 유효 + 층 보정으로 데미지 비례 상승
6. **직업 + 양손무기 스킬** — 마비/수면으로 무력화 후 다단히트로 마무리
