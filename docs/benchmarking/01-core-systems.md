## 게이지

| 게이지 | 위험 수치 | 효과 |
|--------|----------|------|
| 생명력 (HP) | 0 | 사망 |
| 정신력 (MP) | 부족 | 스킬 사용 불가. 일부 마법은 HP 소모로 시전 가능 |
| 배고픔 | 450+ (배고픔) | 체력 자연 회복 중단 |
| 배고픔 | 600+ (굶주림) | 일정 시간마다 HP 감소 |
| 피로도 | 450+ (피곤함) | 정신력 회복 중단 |
| 피로도 | 600+ (탈진) | 정신력 회복 중단 + 최대 체력 점점 감소 |

## UI 버튼

**우측**: 아이템 제작/분해 / 인벤토리(+상태창) / 주문 목록(+등록) / 원거리 무기 / 등록 주문 / 무기 스킬 / 직업 스킬

**좌측 하단**:
- **대기** — 턴 넘기기
- **탐색 모드** — 숨겨진 함정·통로 탐색. 턴 2배 소모, 배고픔·피로도 빠르게 증가

## 스테이터스

| 스탯 | 주요 영향 |
|------|----------|
| 힘 | 공격력 + 아이템 착용 요구치. 힘 부족 시 명중/회피/턴 패널티 |
| 체력 | 레벨업 시 증가. 최대 체력 관여 |
| 민첩 | 회피율 + 이동 속도. 회피율 = 민첩 × (1 + 민첩 × 0.05)%. 주변 빈 타일 많을수록 추가 상승 |
| 지능 | 정신력 최대치 + 마법 효과 |
| 운 | 마법 제외 모든 효과 발동 확률 |

## 장비 슬롯 (총 6개)

| 슬롯 | 설명 |
|------|------|
| 원거리 무기 | 활 등 |
| 오른손 (주무기) | 근거리 1st 무기 |
| 왼손 (보조) | 부무기 or 방패. 이도류 가능 |
| 옷 (방어구) | 경갑 or 중갑 |
| 악세사리 × 2 | 장신구 |

- 착용한 아이템은 인벤토리 칸 차지 안 함
- 격투무기·양손무기는 두 슬롯 모두 사용

## 상태이상

| 상태이상 | 효과 |
|----------|------|
| 중독 | 지속 데미지. 물 속에서 2배 빠르게 해제 |
| 불붙음 | 높은 지속 데미지. 주문서·천옷 소각 가능. 물에서 즉시 해제 |
| 부상 | 지속 데미지 + 행동 속도 감소 |
| 수면 | 행동 불가. 공격받으면 해제. 수면 중 피로도 감소 |
| 마비 | 데미지 1회 + 행동불가 (1턴) |
| 빙결 | 데미지 1회 + 2턴 이동불가 → 느려짐. 근접 공격한 쪽도 느려짐 |
| 느려짐 | 이동 속도 감소 |
| 저주 | 랜덤 능력치 1 감소. 반영구, 중복됨. 정화 물약/맑은 샘/정화·디스펠 마법으로 해제 |
| 축복 | 행운 +2 (유지시간) |
| 실명 | 명중률 대폭 감소. 주인공 인식 불가 |
| 공포 | 도주 |
| 기절 | 행동 불가 (짧은 시간) |
| 피곤함 | 피로도 450+. 정신력 회복 중단 |
| 탈진 | 피로도 600+. 최대 체력 점점 감소 |
| 배고픔 | 배고픔 450+. 체력 회복 중단 |
| 굶주림 | 배고픔 600+. 일정 시간마다 HP 감소 |
