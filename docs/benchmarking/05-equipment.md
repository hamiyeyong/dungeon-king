## 가방

모두 3/8/13층 상점에서 500/750/1000골드. 드롭 없음 → 최우선 구매.

| 가방 | 보관 대상 |
|------|----------|
| 재료 가방 | 제작 재료 (시작 보유) |
| 주문서 가방 | 주문서·고대 주문서. 소각 방지 |
| 연금술 가방 | 씨앗·시약 재료·빈병·이슬병 |
| 보조 가방 | 모든 종류 (주문서·연금술 가방 있으면 해당 종류 제외) |

## 무기 공통

- 강화 시 요구 힘 감소 + 공격력 증가. 고티어일수록 강화 폭 큼. 최대 +10강
- 인챈트 + 아티팩트화로 강화 수치 계승. 티어 낮은 장비는 아티팩트화 후 고티어에 바르기 가능
- **힘 부족한 무기 착용 시 행동속도 저하 → 2연타 피격 위험**

## 내구도 관리

- 수리도구: 내구도 회복, 최대 내구도 소폭 감소
- 장비 강화 주문서: 최대 내구도 완전 회복 + 강화 +1
- 최대 강화 (+10 / 장신구 +5) 도달 후에는 수리·강화 불가 → 최종보스 전까지 9강으로 보유
- 아티팩트화 시 새 장비로 재생성 → 내구도 초기화
- 격투무기는 내구도 소모가 극히 빠름 → 광석 적극 수급 필수
