# 던전 & 맵

[← 로드맵](README.md)

## 기본

- 1층~31층 구성. 5층마다 보스 등장 (보스 처치 후 다음 층 진입)
- 이동: 타일·문·벽 클릭. 1칸 이동 = 1턴 소모
- **회피율**: 주변 빈 타일 많을수록 상승 → 개활지 전투가 유리

---

## 특수 타일

| 타일 | 효과 |
|------|------|
| 물 | 불붙음 즉시 해제. 중독 해제 가속. 번개 공격 범위 확대 + 데미지 증폭 |
| 흙 | HP·MP 자연회복 속도 증가 |
| 절벽 | 뛰어내리면 다음 층으로 (3~18 데미지) |

---

## 문

| 종류 | 특징 |
|------|------|
| 일반 문 | 클릭으로 개폐. 인간형 몬스터만 열 수 있음. 불에 소각 가능 |
| 잠긴 문 | 문열쇠 필요. 화염으로 태울 수도 있음. 열쇠는 반드시 같은 층에 존재 |
| 두꺼운 철문 | 같은 층의 트리거로만 개방 (트리거가 철문 안쪽에 있을 경우 윗층 절벽으로 진입) |
| 비밀 통로 | 가까이 가거나 탐색 모드에서 발견. 횃불 근처 3칸 이내에서 주로 발견 |
| 무너진 벽 | 곡괭이로 파괴. 개수만큼 곡괭이가 같은 층에 존재 |
| 바리케이드 | 폭탄·화염 물약·화염구·횃불로 제거 |

## 오브젝트

| 오브젝트 | 효과 |
|----------|------|
| 알림판 | 도움말 + 확률로 골드·횃불·재료 |
| 트리거(레버) | 철문 개방 |
| 항아리 | 확률로 골드·아이템 |
| 해골더미 | 확률로 골드·아이템. 지나갈 수 없음 |
| 제단 | 장비 또는 주문서 |
| 관 | 확률로 아이템 or 몬스터 소환 |
| 무덤 | 확률로 아이템 or 유령 소환 (동시 가능) |
| 큰 무덤 | 거대 유령 소환 가능 |
| 상자 | 골드·아이템. 가끔 미믹 |
| 고급 상자 | 잠금. 열쇠 필요. 귀한 아이템 |
| 유리 상자 | 안이 보임. 열쇠 1개로 선택해서 열기 |
| 지식의 석판 | 미확인 아이템 1개 무작위 식별 |
| 책장 | 주문서·물약 정보 획득 |
| 마법사 석상 | 영웅의 돌 사용 → 마법 강화·습득 |
| 전사 석상 | 영웅의 돌 사용 → 장비 +1 강화 |
| 천사 석상 | 랜덤 버프 (HP·MP 회복 / 스탯 증가 / 아이템 축복 등) |
| 악마 석상 | 랜덤 (음식·아티팩트 지급 or 흡혈·상태이상) |
| 이상한 우물 | 아이템 던지면 변환 (같은 종류 다른 아이템 / +1 강화 / 아티팩트화). 스탯 물약도 변환 가능 |
| 오염된 샘 | 마시면 HP 회복 or 중독. 정화 물약 넣으면 맑은 샘으로 변환 |
| 맑은 샘 | 마시면 HP·MP 회복 + 배고픔·피로도 250 감소 + 모든 상태이상 해제. 물약 던지면 효과 확인 (미확인 유지) |
| 야영지 | 나무 1개 + 화염초 꽃잎 or 횃불로 점화. **요리 먼저(무한)** → 휴식 1회(불 꺼짐, 피로도 초기화 + HP·MP 회복) |
| 연금술 흰 솥 | 빈병 + 재료 2개 → 물약 (발현도 높은 재료 기준) |
| 연금술 검은 솥 | 빈병 + 재료 2개 → 물약 (발현도 낮은 재료 기준) |
| 환영의 석판 | 전통음식·만나 드롭 |

> **야영 주의**: 휴식 후 낮은 확률로 나쁜 이벤트 발생 (몬스터 등장 / 골드·재료·주문서 도난). 귀한 아이템은 바닥에 내려놓고 자는 것이 안전.

## 약초밭

밟으면 효과 발동 + 확률로 약초 획득. 씨앗을 심어 생성 가능. 아이템 던지면 강제 발동.  
**함정 위에는 약초가 자라지 않음** → 함정 위치 탐지에 활용 가능.

| 약초 | 밟을 때 효과 |
|------|-------------|
| 얼음송이 | 빙결 (데미지 없음) |
| 피이끼 | 데미지 + 마비 |
| 산삼 | 즉시 체력 회복 |
| 나이트쉐이드 | 데미지 + 주변 독가스 살포 |
| 암브로시아 | 40AP 동안 해당 지역 유닛 무적 |
| 영지버섯 | 일정 시간 체력 회복 (이동 시 효과 상실) |
| 만드라고라 | 정신력 회복 (가끔 데미지) |
| 화염초 | 불붙음. 고기·씨앗 던지면 요리·점화 가능 |
| 꿈결초 | 데미지 + 수면. 피로도 감소 효과 |
| 마늘 | 모든 상태이상 정화 |
| 수풀 | 밟으면 자동 소멸 + 확률로 이슬·주문서·물약·빈병 드롭. 불 옮겨붙음 |

## 함정

| 함정 | 효과 |
|------|------|
| 스파이크 | 데미지. 영구 작동, 아이템으로 해제 불가 (해제 시도 후 회수하려면 밟아야 함) |
| 독가시 | 중독 데미지 |
| 부상 | 부상 상태이상 |
| 독 구름 | 가스 퍼짐. 진행도에 따라 범위 증가 → 멀리서 물건 던져 처리 |
| 마비 가스 | 가스 퍼짐, 마비. 마찬가지로 멀리서 처리 |
| 화염 | 불붙음 |
| 얼음 | 빙결 |
| 텔레포트 | 같은 층 내 랜덤 이동 |
| 전격 | 주변 칸 데미지 |
| 소환 | 적 1~3마리 소환 |
| 알람 | 근처 적에게 플레이어 위치 알림 |

> **탐색 모드**: 턴 2배 소모로 주변 숨겨진 요소 확정 탐색. '예지의' 인챈트 장신구가 있으면 비중 크게 감소.
